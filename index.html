HEXO进阶

Posted on 2025-09-29

HEXO接近是最近有一些朋友提出的问题，然后我做了总结，如果你也在使用HEXO，不妨看看，应该会有些帮助。

  • 1、博客部署样式出问题了怎么办？
  • 2、电脑重装或者误删了本地博客怎么办？
  • 3、想使用两台电脑写博客怎么办？
  • 4、为何使用百度搜不到我的博客？

使用Jekyll解决前三个问题。

不得不说 Jekyll 确实可以解决我上面三个问题, 因为 Jekyll 是直接把Markdown格式的文章直接放在github仓库里的, 相当于直接用git来管理博客了, Github 官方也很推荐 Jekyll 。 你可以先看下 Jekyll 搭建博客的voyagelab, github地址, 当然了这只是很普通的, Jekyll 也有很多主题可以选择的, 更详细的请看Jekyll中文文档、Jekyll英文文档、Jekyll主题列表。
在 Jekyll 上逛了一个星期的我又回到了 Hexo , 发现 目前 Jekyll 对主题和一些插件的支持相对 Hexo 来说, 没那么友好, 可能有一些其它的方法只是我没找到而已,关于 Jekyll 搭建博客就介绍到这, 如果有问题的话可以评论, 或者联系我。

使用Hexo解决上面前三个问题

是的, 我大Hexo同样可以解决上面三个问题, 那就是使用git。关于如何使用 Hexo 搭建博客请看我另一篇文章HEXO搭建个人博客, 如果搭建的过程中出现了问题, 我们可以交流交流。现在我假设你已经能基本使用 Hexo 了, 接下来就看看如何来管理博客。

使用git管理博客

Blog-Growing是我管理博客的仓库地址。结构大致是:

– Blog-Growing
　　|– .git
　　|– .gitignore
　　|– Hexo
　　　　| ..
　　　　| ..
　　　　| 整个博客的配置信息

你可以把我的的Blog-Growing clone下来, 看看一些基本配置是怎么配置的。需要提醒的是, 要把主题下的.git给删除掉 , 当然了你也可以使用 git submodule 来管理, 具体的管理方法我这里就不做多的介绍了, 想了解的话可以私信我。
上面的过程基本做完后, 怎么来解决我开篇提的前三个问题了。

具体实现:

一：家里电脑使用博客
　　建立git远端仓库管理博客,并使用家里的电脑把本地博客的配置推送到远端仓库。
二：公司电脑使用博客
　　到了公司只需要执行sudo npm install -g hexo,然后cd到你的博客目录下,如我cd 到Hexo目录下, 然后执行 hexo server 就可以在本地预览博客了。
三：使用Git保存
　　修改好博客后记得先使用git来提交下, 即使下次把博客的样式修改坏了, 也可以使用 git reset --hard 来回退。如: 我cd 到 Blog-Growing 目录下使用git提交。
四：博客提交
　　1、修改好的博客使用 hexo d 展示到博客页上。
　　2、git push 整个本地博客。

提示: 在这里 git 仅仅只是用户做博客的版本管理的, 博客的样式修改、基本部署还是使用 hexo 来操作的。

让百度能搜索到你的博客

为什么要使用百度搜索？

　　有人可能会说作为一个开发人员, 你不会用 Google 啊。 是的, Google是能搜到我们搭建在 Github Page 的博客, 会用Google 也是一个开发人员必备技能之一。但是, 我们生活在天朝, 所以百度还是总有会用到的时候, 或者是你想让更多的天朝人能搜到你。

为什么使用百度搜索不到 Github Page 上的博客？

有人联系过 Github Support 部门 , 给出大致的意思就是: 百度爬虫爬得太猛烈，已经对很多 Github 用户造成了问题。所以 Github 将禁止百度爬虫的爬取。

如何让百度能搜索你的博客?

　　根据上面说的, 目前发现只是Github Page禁止了百度搜索, 所以让百度能搜索到你的博客还是有一些方法的。例如:

  • 自己搞个VPS,博客部署在VPS上。
  • 博客部署 Coding.net 上, GitCafe已经合并到 Coding 。
    我使用的是第二种方法, 博客部署在 Coding.net 上也相对简单些。

在Coding上部署你的博客。

　　Coding同样支持Hexo、Jekyll等博客的部署, Coding 跟Github还是挺像的,而且是中文。 同样的在Coding里面建一个项目,项目名字跟你的用户名一样,这里我就不啰嗦了, 说几个需要注意的地方:
注意一:
　　在Coding Page 上部署博客,需要把博客推送到coding-pages 分支上, 分支名字是固定的。
注意二:
　　Coding Page 不支持自定义CNAME, 你需要点击到Page模块,然后添加一个域名来绑定。

更详细的请看Coding Pages 官网介绍.

为什么我不推荐你读博士

Posted on 2025-09-25 Edited on 2025-09-29

最近几年兴起了一股。大众读研热。每一个学校的本科生。在大三下到大四上的那个暑假，都会奔走于各个高校之间，参加他们的保研夏令营。或者是其他名字，但本质一样的活动。这些学子无一不像孔雀开屏一般展示着自己的种种能力。分享着自己在本科期间的。种种的工作。只为得到。对应院校老师的青睐，获得一个研究生的录取通知。可是如果你问他们为什么要这么做？他们只会用特别清澈的眼神来回答你，因为大家都在做。在这条人声鼎沸的赛道上，每一个人参与者保研或者考研的竞争。大家一起追逐着更高的学历。你期望有更好的未来。可是如果你细细追问的话，为什么大家都在做的话，你也需要做。被问之人往往打不出个所以然来。只能在口中泛泛地讲出。读研之后，我的薪资水平会更高。读博之后，我的未来发展平台会更高。反正在没有毕业之前,没有真切的体会到读研究生给自己带来的好处，给自己人生带来的助力。已经给自己画好了一个充满光明的。康庄大道。那么。存在于这些学生脑海中的康庄大道，光明未来。究竟。与他们所期待的一不一样呢？我觉得这是一个问题。需要被好好疼。

首先是关于薪资问题。以计算机行业为例，一个本科生毕业的薪资和一个研究生毕业的薪资相差是很大的。本科生的薪资最多也是年薪四十万左右。以硕士研究生为例，薪资可以达到五六十万甚至于七八十万。博士研究生毕业之后薪资七八十万到一百万都大有人在。所以无数学生都为了这个理由挤破脑袋。进去读研究生。可是这些让人看到的特别高的高薪，本质上不过是一种幸存者偏差。因为只有拿到特别高的高薪的。研究生才会将自己的高薪主动的爆料出来。而那些没有拿到特别高的高薪的。肯定。大概率也许比本科生的薪资更高吧。但这又涉及到另外一个问题。就是性价比问题。是这样一种情况，如果每年复读都能提高二十分，有多少人会坚持下来呢？不是说多读一年，对于自己的分数提高没有意义。而是多读的这一年。对于自己人生的助力。和自己失去的代价？在度个天平上是否相等的？
自己放弃了自己最美好的青春年华。在那样一个充满活力，身体健康，思维敏捷的黄金十年中，干的只有实验室中的那些实验，每天不仅要承受极大的压力，在导师的催促声中，在能不能按时毕业的压力中，在众人对于你的期许中，能不能达到自己的那个彼岸？
现在的中国人一生都在追求上岸，因为环境的不确定性，导致他们极度渴望安全感，这种安全感能够让中国人在暴风雨来临的时候平安无事，仿佛圣经大洪水中的诺亚方舟。因此，中国人极力美化彼岸，极力追逐登上彼岸。可这世上真的有彼岸吗？只有一种境况和另外一种境况的不同罢了，本质上都是海洋，只是这片海洋暂时平静，那片海洋波涛汹涌罢了。且看大禹治水，疏堵结合，最后人定胜天。中国人从来就应该是健壮的水手，在海洋中遨游，或者乘风破浪，一往无前，或者仰面调息，暂时休息。这样才能确保自己的永久安全。这种安全从来只会来自于自身的力量，而不是外界的保障。

即使有了杰出的成果，在三十岁之后事业有成，在职业岗位上呼风唤雨，在北京成家立业，可是这真是自己原来想要的生活吗？《明朝那些事儿》的最后，写的是徐霞客，为什么用徐霞客结尾呢？因为所有的宏图伟业，英雄事迹在百年之后不过是过眼云烟。纵情山水，躲开世间的纷繁杂乱，摒弃那些功名利禄，实实在在做自己，欣赏这世间的美丽，不也是一件好事？平平淡淡地过着自己地小日子，做一个市井百姓，庸人又何尝不可？

再者，有多少人喜欢科研？能在日复一日地实验中保持热情？这世上大多数人是不适合读博士的。毕业要求高，五年六年生活枯燥，在日复一日的劳动中，又有多少快乐呢？扪心自问，自己真的喜欢研究生的环境和氛围吗？大部分人都是选择硕士不是博士就已经能够说明问题了。大家都不喜欢，都想呆三年就赶紧走，不想多留一刻。这就是现实。只有极少数人喜欢科研。

不是说读博士没有收益，而是这个收益和自己付出的比率是否合适？我觉得读博其实低性价比选择。

，没有不熟悉的领导；5.笔杆子自荐（小圈子内部也需要有靠谱的支撑人才，领导最需要的就是给上级汇报工作，笔杆子也是一条路。小地方的小国企还有机会，大单位已经卷爆了，人才过剩的同时AI给了笔杆子致命一击，这条路也不好走。）第二种：伪装成企业的行政部门，这种企业事实上是某个行政机关的外围，比如行业内某些认证服务；特殊的行业产品制造企业，比如某些消防，环保产品；网络安全等级保护认证评估部门；计量单位等等这种企业既有行业的专业性，也有行政机关的权威性（可以狐假虎威），有能力可以活得很好，有关系也会有很适合的位置。这类企业通常没有特别大的同行竞争，而且还有准入限制，通常还会有牌照要求。在这种企业论资排辈应该是常态，也存在一定竞争，但竞争压力要比其他类企业小很多。第三种：伪装成企业的金融机构，这种企业规模大，但是两头在外，上游和下游均是完全市场化竞争的状态，国企依靠自身优势做规模，依靠低利息带来的低成本和同级竞争，同时通过为上下游隐形放贷而获取订单和利益。比如各种国企贸易公司，弄了大雷的钢铁贸易，看似在做贸易，其实是在放贷；在某个产业链中的大体量国企，看似是一个高新技术企业，其实是通过国资背景为上游公司签大订单合同，帮助上游贷款而获取廉价原料，通过自己的低息贷款为下游长账期供货从而提高产品售价，这样可以低成本采购原材料，更高价销售产品，利润除了产品本身以外还有放贷的收益；更典型的是民航，民航看似是交通运输服务业，其实是一个金融机构，上游的飞机和航材是进口的，下游的旅客是要在市场中不断竞争的，两头在外，通过自身的资金优势不断积累资本和市场。这种企业通常体量巨大，年营业额至少百亿起步，通常在数千亿左右，规模大的近万亿。在这里混，能力非常重要，所做的业务是隐藏在复杂的行业之中金融操作，一方面要有懂特定行业技术的专业人员，另一方面也要有懂资本运作的人才，如果要更进一步，两个方面都要懂，对个人能力的要求很高。虽然这种企业的资金流水巨大，但监管力度不像第一类国企，同时由于两头在外，企业有大量可用的漏洞，掌握了这些漏洞几乎可以合法的捞钱。因此这种企业的关系也非常重要，各个漏洞必须由核心圈子的人来控制，核心岗位必须由核心圈子的人占据。在这种企业，光有能力不能到核心岗位，光有关系核心岗位的工作压力接不住。从世俗利益出发，这种企业值得努力去往上爬，不惜钩子的往上爬。第四种：伪装成企业的民政部门，这种企业为老百姓提供廉价的基础服务。比如邮政的普通包裹，铁路的绿皮车。属于国家的基础服务，不光不考虑盈利，国家还要给予补贴。各地的地铁公司也是这种国企。这种企业通常体量很大，由于要拿大量的财政补贴，管理体系也异常健全。在这种企业混，能力的作用一般，比如地铁公司，一切操作都有完善的规章制度，你一个牛马能干啥？想要出彩，没有上层领导的支持基本不可能，但是一旦出问题，你的职业生涯基本到此为止了。牛马们在这种单位首先要学会自保，任何一个事故都会把你焊死在底层一辈子爬不起来。作为一个面向大众的民政部门，要求成员完全按照操作规程履行职责，邮政，铁路，地铁，公交都是此类，对个人的能力要求极低，由于企业自身的运营必然是亏损的，底层的牛马待遇通常不会太高。同时这种企业由于体量巨大，内部有复杂的利益集团，在上级的强监管下，没有关系很难进入这个圈子。这种企业关系远比能力重要。从世俗角度来看，值得往上爬，虽然不及前三类，但也足够稳定。第五种：伪装成企业的jun队，当年某一任最大的大哥要求XX一律不得经商，因此jun队只能依赖财政拨款存在，这种企业也是类似，他的订单完全依赖财政，产品只有唯一的toG客户，虽然看起来搞的热火朝天，分析其资金流，主要来源甚至是唯一来源就是财政拨款。和老百姓日常关系不大的航空航天，兵器船舶等企业就是此类，和老百姓日常有点关系的比如之前印刷发票和火车票的企业（差不多快完蛋了）。这种企业业务来源单一，受单一甲方影响极大，发展前景同样受甲方影响极大。（1）如果所处行业刚好是某项国家重点卡脖子攻关目标，那么上级领导的关怀和海量的经费将源源不断，此时能力出众但没关系也可以很快出头，毕竟整个体系都在扩张，十分缺有能力干活的人，如果有关系且是甲方的关系那前途无量，只有企业内部的关系也可以干很多出彩的工作，进而掌握核心岗位。和百姓生活关系密切的芯片，刚刚被处理完的芯片大基金就是这种。和百姓日常生活关系不大的航母也是这种。（2）如果行业属于国家必须，但技术已经成熟的行业，比如兵器集团下面的某些常规五器制造公司，航空航天部分低端生产企业，这类公司，订单需求长期稳定在某个区间，没有太大的增长空间，技术创新的空间也极其有限（典型产品比如步抢，手抢你还能玩儿出什么花儿来？），而且甲（音jun）方隔三差五派人来审计，变着法要求降价，企业的资金就会很紧张，降本增效挂在嘴上，领导需要削尖了脑袋从上面弄钱，内部形式主义极强。这类国企没关系会很不好过，一大堆垃圾岗位，写不完的项目申请材料（最终项目没有下文），加不完的班（鸡蛋里挑骨头），降不完的成本（除了领导的收入，其他牛马的收入都可以降，工作时间都可以长），弄不完的荡锏（没有可出彩的工作，关系户只会搞这个）。即使你很有能力，也很难干出成果。即使领导胡乱搞，企业靠体量和历史地位一时也垮不了。这种企业由于行业大势已过，但也不至于很快被时代彻底抛弃，曾经也辉煌过，是落魄的秀才放不下架子，身上的担子很重。内部关系错综复杂，可能祖孙三代都在里面。如果没有关系，最好攒一点经验就跑，换一个处于上升期的行业。如果由于种种原因没法跑路，最好找个副业，因为这种大而老的企业，没关系的外人进去，即使把你放到核心岗位你也很难做事。如果硬要往上爬，参考上文混小圈子的办法，女婿党，秘书党二选一。说实话，花这么大精力在这种企业往上爬，有点不值，有这种想法赶紧换个好点的地方往上爬。（3）如果行业只有一个toG的甲方，但甲方选择新的技术，或者放开了市场竞争，引入民间资本或其他国企，这种企业的生存与毁灭完全看几任领导的决策。远的比如通信设备行业，稍微近一点的比如印火车票和发票的企业，最近的比如商业航天。一旦没有了甲方的庇护，民间的竞争力会瞬间袭来。华为早期做电话交换机的时候，短时间吃掉了国内很大的份额。火车站可以刷身份证以后。印刷火车票的企业迅速衰落。电子发票出现以后，传统发票时用量极速减少。目前的商业航天还处于孕育期，一旦某几个型号成熟稳定，国家一定会放开竞争，到时后除了五器和载人，大量的载荷会流向商业航天。而航天自身也不争气在当前这个时间点出了一系列问题。在可预见的未来，十年内，某些老航天单位将会和造手榴弹的一个地位：不能没有但也不能上桌。这种企业的发展完全看领导的能力，而且主要看一把手的能力。本身企业是即将或已经被丢出温室的花朵，如果一把手准备混日子等退休，企业死路一条。如果领导有市场竞争经验，有调整公司结构的思路，还有希望。在这种企业混，如果公司体量大，比如有大几万人，还能混一段时间，运气好可以混十几年。毕竟体量大，上级会考虑影响，实在活不下去了总会给点补贴，但不要指望有多少，最多够活。此时个人能力不重要也很重要，这种半死不活的企业你根本施展不开手脚，但公司体量又在这，维持日常运转还是有办法的，这时聪明且没关系的人会不断跑路，聪明且有关系的人跑的更快甚至根本不会来，有关系的人大多是一些能力一般甚至很差的人，这些人如果做领导，下属当中的聪明人会跑的更快，大领导再蠢也不见得会大量提拔这些关系的人，而且这种企业内，很少会有硬关系的人。这时如果你作为一个没有背景的聪明人，只要你熬得住，总会有机会的。等你当了中层领导就会发现，下属都是傻逼，但都和你的上级或同级有着千丝万缕的联系，赶又赶不走 ，骂又骂不得，稍微复杂的活是一点干不了，导致什么事情都得你自己亲力亲为，当领导比当牛马还累；如果你熬了下来并有机会更进一步，你就会发现公司早已资不抵债，靠财务的复杂操作维持局面，各种大雷都在等着爆炸，每天的工作都是弄钱和排雷，你身心俱疲感觉公司随时都会完蛋，可是公司活了一年又一年，你想做出某些改变，发现即没有人也没有钱，随便一个风险就会让公司的雷爆炸，思来想去只能求稳。如果公司体量小，比如只有一两万人，或只有几千，主要看一把手，年轻、有能力、有思想、具备个人魅力和对上能力缺一不可。除非是这种领导，其他类型的领导越早跑路越好，跑的早还能卖个高价，跑晚了要饭都没门了。此时能力完全不重要，本身就是要被淘汰或者被开源的技术，有关系也只能比其他人多苟两三年，最多五六年。尽早跑路是关键。如果公司体量更小，只有几百人甚至几十人，赶紧跑，在这样企业工作的兄弟你在看到这篇文章此处的时候就赶紧找下家，发完简历再往下看，这种假国企呆一秒都是对自己的惩罚。在第五类公司混，核心是要看自己所处的公司是御林军，万岁军，边军，还是上了裁军名单的杂牌军。如果行业已经被国家放开，或者已经被新技术替代，这种国企的未来必然是滑滑梯。比如历史上做胶片的乐凯集团，从央企一级公司混到被合并；汽车行业从一汽、二汽到资质大规模放开，汽车行业央企干不过国企，国企干不过私企。第六种：伪装成国企的私企，这种企业因为历史原因提供某种服务和产品，既没有垄断地位，也不具备先进技术，凭借历史积累和国企背景存在。最典型的是各个地方的设计院，看似一个个都是国企，卷的比集中营还可怕。这种国企数量最多，最具有迷惑性。地方国企，三、四级央企中存在大量类似企业，在里面熬的兄弟也最痛苦，国企的优点一分没有，私企的内卷一分不少。这种企业有脑子的关系户绕着走，某些笨蛋关系户实在找不到工作，家里只能安排到这儿工作，业务做不了只能去办公室、财务部门、挡箭部门、后勤保障部门，最终除了负责业务和技术的部门，其他部门都是关系户。业务条线和管理条线之间有巨大的鸿沟。业务口的都是各地来的牛马大学生，管理条线都是各种关系户。在业务条线上可以凭能力混个位置，在管理条线就安心做牛马。在这种企业混，首先要想明白这个企业还能活多久，摸清楚所在行业的情况，和上级国资委怎么看这个企业。无非这么几种情况：（1）行业已完全市场化竞争，市场规模仍在扩大，企业自身也完成了市场化改造，利润尚可，上级评价不错；（2）行业已经完全市场化竞争，市场规模仍在扩大，企业管理落后，零利，微利甚至亏损，上级要求企业做出改变；（3）行业已经完全市场化竞争，市场规模缓慢增长，企业管理落后，连续小幅亏损，上级强烈要求企业做出改变；（4）行业已经完全市场化竞争，市场规模在萎缩，企业管理落后，连续大幅亏损，上级对企业管理层不满，并不断空降领导；企业的寿命有三个关键因素：目前的体量，市场的规模和状态，企业自身的管理水平。第一步考虑是否跑路，如果所在行业仍然在增长，可以不跑路；如果开始萎缩，看上级时否投新业务过来，如果没有，一定要跑，国企在这种局面下是竞争不过市场的；如果国企年纪很大，且市场萎缩，尽早跑路，上级其实也在等老登们退休，好把企业关掉。第二步考虑是否全力以赴投入工作，如果企业管理较先进，有一定竞争力和影响力，可以投入全力；否则混日子即可，专注于副业比你加班的工资高的多。第三步考虑是否往上爬，如果行业有一定的技术门槛，建议你先打磨自己的技术水平，除非有好大哥赏识，不要浪费时间往上爬，这种国企不见得比私企稳定，自己的技术才是最稳定的保障；如果行业没有技术难度，全靠国企的牌子接活儿，比如国资背景的施工单位，往上爬就比较重要，更高一级的工作内容，待遇和接触的对象都有质的改变。第四部考虑怎么往上爬，有好大哥赏识就玩儿命干，没有参考混圈子的五法。在第六类公司混，能力非常重要，这里的能力不是什么工作能力，是认清公司本质，并做出正确判断和选择的能力。六类国企犹如六道，上三道天、人、阿修罗；下三道畜生、地狱、饿鬼。（君宫口单位多数是第五种，也有不少第六种和第四种，前三种极少，而且非常难进）愿以上拙作，帮各为兄弟厘清思路，做好选择。

今天5点钟，我到了西安站。那个时候天还没有亮，夜色笼罩在了西安墙上。等到我出西安站的时候。朝霞微信。我顺手拍了一张照片，感叹自己很久都没有见过这样的早晨了。我提着箱子。一直朝南走。走到了五五路口，便搭乘地铁回到了创新港。我的保研之旅应该就此结束了。接下来无论是本科直接就业还是考研的话。都是我的个人的一个选择。或者其实还有一个选择是在九月份的时候去捡漏。无论如何。今天我是狠狠地休息了一天。看了看变形金刚和lck那边的比赛。

在刷知乎的时候我看到了一句话很有意思。在这里我想谈谈那句话，是说不要过分苛责年轻的自己。因为当时的自己。也很困难。不要和当时的别人。一起苛责当时的自己。到时候自己。经验和认知程度，也就是那个水平。在那么大的压力环境下即使犯了一些错误也没有必要。每天夜里都回想起来。每天夜里都不断检讨自己的错误。这不是反思，是苛责。是啊，当时的我没有任何经验。又处在高三年级那么大的压力环境下。再加上从小养成的那种偏执的性格。无论干出什么事情。都不应该再多责怪当时的自己了。与当时的自己，也已经很难了。我还记得在贝斯林华说爱而不得的时候，那种无力感。在人家看到我头也不回。的时候的那种？无奈。广东能够很多事情已不愿意再回去。也不愿意再回忆。因为一切的一切都已经结束了。我接下来也要面临更加痛苦的选择。我没有来自父母辈的帮助，也没有很顺畅的信息渠道。我只能凭借着我平时互联网冲浪时获得的有限信息来进行决定。而面对快速变化的行业，获得有限的信息，类似于盲人摸象。这时候我只能进入我昨天说的一句话，其实是一个错误的决定，只要被得到坚定的执行，也可能出现不一样的效果。我觉得自己这句话也是很有哲理的。无论如何做的决定，就不要再改了。坚定的投入资源。是本科直接就业还是保研亦或是考研？都只不过是人生的一个阶段儿。人生的容错，超乎你的想象。只要我不放弃，不气馁。确定好自己的目标，努力的往前走。谁说不能够走到终点？

一些关于生活的思考

关于自己生活质量的真实反映。究竟是别人眼中自己生活的好坏，还是自己实打实地感觉到生活的好坏？人的痛苦呀，除了肉体上实打实的痛苦之外，所有的痛苦都源于自己的大脑，都源于自己的认知。也就是说，只要精神经历的不断，改变自己的痛苦也会不断改变。比如说一个收入低下的搬运的环卫工人。他的最大的痛苦可能就是自己的收入不够自己生活的进一步提升。只能维持到一个较低的生活水平。而一个投行的。量化交易员。他的痛苦可能就是生活中的种种不顺，比如自己的学子女学习不够，好比如自己的初恋没有找到，比如自己的老婆不够漂亮，比如自己最近的情人，又问自己要了什么东西，又和自己闹了矛盾之类的。这些痛苦其实就源于自己生活中的种种不满。

还有就是别人认为你过得好和别人认为你过得不好，与自己真实生活过得好，和过得不好。究竟哪一个比较重要？我想大部分人都会清晰的认知到自己过的真实生活，好是最重要的。但是很多人潜意识中还会觉得只有过程令人别人羡慕的生活才是真正的好的生活。有豪车，有大别墅，有名车名表，有漂亮的老婆，有一份薪水高的令人羡慕的工作。这些东西统统都会让别人羡慕，也就是十足意义上，大部分人一生所追求的。可是有一个小小的问题，大部分人所追求的真的是他们所想要的吗？就比如说吧，大部分人都愿意在大城市里面安家落户。从牺牲掏空了六个口袋，在大城市里面买了一套一百多平米的房子。忍受着每日超远距离的交通，忍受着职场的各种高压，忍受着无休无止的恶劣的职场文化。糟糕的空气，不安的环境。以及对子女阶级滑落的担忧。种种因素影响下的生活，真正是他们刚进入这个大城市里面所期望的。很多人在小城市里面常常会羡慕大城市的生活，却不知道大城市里面的人也常常会羡慕小城市里面的宁静祥和慢节奏。所以究竟是城市里面的生活好呢？还是小城市生活好呢？其实并没有好与不好，只有自己个人的选择罢了。喜欢大城市的繁华，喜欢大城市的高工资，喜欢大城市的物资的充盈。教育医疗资源优渥。那我就去大城市。喜欢小城市里面的宁静祥和，喜欢他们的慢节奏，喜欢他们的物价低，喜欢他们的。有很多熟悉的朋友。可以和自己喜欢的人相携手度过一生。那你去小城市。所以我们到底追求的是什么呢？追求的是大城市或者小城市嘛。显然不是我们追求的，是自己幸福的生活。那么到底什么是幸福的生活呢？在我看来，无非是有一个爱自己，自己爱的人。有一份能够为你生活提供充足的物质条件的工作和事业。仅此两项儿。我呢，显然很大概率可能不会拥有第一项了，那么我就需要好好拥有第二项追求第二项的。把握好这个机会，为自己三十岁攒下人生的第一桶金。到三十岁之后，凭借自己的聪明和才华，凭借自己的胆大心细，我相信自己绝对不是一个庸人。绝对能够。创造出。一片属于自己的天地，赢取属于自己的那份财富。那么初到这里我们便可以轻松得出。别人眼中自己过得怎么样，其实并不重要，关键在于自己真实的生活究竟是不是那样。别人怎么看你并不重要。别人怎么评价你的事业并不重要。别人对你有什么样的认知也不重要，因为别人终究是别人，他影响不了你。坚持自己内心所想，继续走下去，继续干下去便可以了。至于其他人，绝大部分人都是普通人平常人，他们对于世界的认知有限，他们对于这个世界的理解有限，他们并不能作出理性而合理的选择。他们只能凭借着自己从生活中得到的一些感性的经验，凭借着自己。所了解到，那些号称古人的智慧，凭借自己从抖音快手或者不知道什么地方所听到的人生证言。来判断，了解他们眼中的世界。这无异于盲人摸象，无异于坐井观天。其实我也是这样盲人摸象和坐井观天，但是我觉得因为我想的够清楚。我追求的东西目标更明确。所以即使坐井观天，我也能够凭借着自己的感觉找到大致正确的方向，走到自己所期望的那个终点。希望是这样吧。

最后呢，我觉得不是所有的人都能够。正确的教育，下一代的子女。有些人认为养育子女很简单，给他一口吃的不太够，饿不死就行了。可是如果要真正的教育自己的子女，如果要真正的教会他如何做人做事，如何看待这个世界，如何理解这个世界，如何从这个世界已有的现象中抽丝剥茧，探清楚规律，来判断这个世界的趋势，开始是一个很难的课题。如何和自己身边的人处理好关系？如何在危机中挽回一段关系？如何处理好自己的关系，如何经营与他人之间的友谊？这些都是非常非常重要的课题。如何表达自我？如何经营自我？如何提升自我？如何认清楚自己？如何定位自己？这些关乎于自己本源的东西更加重要。除了方法论和对自己的认知对世界的理解。还有交汇。一些实用主义的东西。作为一个理科生，天然就是实用主义的簇拥者。所谓那一句，无论是黑猫白猫，只要能抓到耗子就是好猫再正确不过了。
